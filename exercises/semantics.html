Instructions

  • 1. Inspect the HTML structure – it's divs everywhere!
  • 2. Identify where more meaningful tags should be used.
  • 3. Replace the
    s with semantic HTML elements. (Think <header>, <main>, <article>,<nav>, etc.)
  • 4. Review your changes and make sure the structure now clearly represents the content.
  • 5. Test the page with assistive technologies like screen readers and see how much easier it is to navigate!
